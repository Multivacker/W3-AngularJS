When you use the ng-repeat directive to create dropdown lists, the selected value must be a string.

In this example you will have to choose between the color or the model to be your selected value.




When using the ng-options directive, the selected value can be an object: Example Using ng-options:

Select a car:

You selected: {{selectedCar.model}}

Its color is: {{selectedCar.color}}

When the selected value can be an object, it can hold more information, and your application can be more flexible.
We will use the ng-options directive in this tutorial.
